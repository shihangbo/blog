# 个人博客
坚持写完一个blog，发奋图强...

## 项目一期计划
<p>项目的起步阶段需要完成如下功能模块</p>

### 前台功能
1. 首页内容聚合
2. 列表页
  * 分页列表
3. 内容页
  * 评论
4. 注册
5. 登陆

### 后台功能
1. 登陆
2. 分类管理
  * 分类列表
  * 添加分类
  * 修改分类
  * 删除分类
  * 查看分类下所有博文
3. 内容管理
  * 内容列表
    * 所有内容
    * 按分类查看
  * 添加内容
  * 修改内容
  * 删除内容
  * 查看内容下的所有评论
4. 评论管理
  * 评论列表
    * 发布评论
    * 查看指定评论
  * 删除评论

## 项目二期计划
<p>对项目前台进行UI优化，新增模块，新增功能等</p>
